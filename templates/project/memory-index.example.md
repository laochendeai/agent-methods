# MEMORY Index Example

这是一个最小 `MEMORY.md` 示例。它只做索引，不直接存放完整记忆正文。

## Personal Memory Index

- [User Role](user_role.md) - current user role and long-term collaboration context
- [Project Feedback](project_feedback.md) - recurring product or workflow feedback
- [Execution Habits](execution_habits.md) - stable working preferences and routines

## Team Memory Index

- [Testing Policy](testing_policy.md) - shared testing rules for this repository
- [Release Habits](release_habits.md) - shared release expectations and guardrails
- [Architecture Decisions](architecture_decisions.md) - team-level design constraints

## Rules

- 每条记忆都写到自己的 topic file
- 每条索引尽量保持一行，便于进入 prompt
- 发现重复时优先更新原文件，而不是继续增加入口
- 不要把敏感内容直接写进索引或 team memory
